# PremiumTech Ecommerce — Project Instructions

## Stack

- **Frontend**: React 19 + TypeScript + Vite + TanStack Router + React Query + Tailwind 4 + shadcn/ui
- **Backend**: Node.js + Express + Prisma 7 + PostgreSQL (Neon) + Cloudinary
- **Architecture**: Feature-Sliced Design (FSD)
- **i18n**: react-i18next (EN/ES)

## Rules

- Never build after changes (`npm run build` está prohibido)
- Never add Co-Authored-By to commits
- Use conventional commits
